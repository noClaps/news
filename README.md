# News

A minimal Hacker News client.

## Build instructions

1. Clone the repository.

   ```sh
   git clone https://gitlab.com/noClaps/news.git
   cd news/
   ```

2. Install dependencies and start the development server.

   ```sh
   bun install
   bun dev
   ```

3. Build the site.

   ```sh
   bun run build
   ```

   This will place the built files in `dist/`.
